# Alien Isolation - iOS - Source Code Files Breakdown

These will be broken down by category / system.

If the paths get too long, they will be shortened to the section of the path that is relevant, for that category.

The following categories will be covered:
1. Game
2. Engine (CATHODE)
3. Physics (Havok)
4. UI (Scaleform)
5. Video (CRI)
6. Audio (Wwise)
7. Rendering Compatibility Layer (Indirectx - Feral)
8. Filesystem Compatibility Layer (MacDoze - Feral)

> Do note, that the purpose of most of these files is assumed, from either the file path and/or the file name. Alternatively, they will be determined by grouping symbol names, with adjacent appearances of system names.
